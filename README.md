目前版本EPD使用的是Python 2.7。安装完毕后，你将可以用到下面的这些包:
* Python科学计算基础库：NumPy、SciPy、matplotlib以及IPython。这些都包含在EPDfree中。
* IPython NoteBook依赖项：tornado和pyzmq。这些也包含在EPDfree中了。
* pandas(0.8.2版或z更高版本)
